== Description ==

If your page/post titles and page/post slugs do not match, you can still find them by slug in the Admin Dashboard. In order to activate Search by Slug feature, in the search field simply start your search with "slug:", for example, slug:contact.

== Installation ==

1. Upload `sbs` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
